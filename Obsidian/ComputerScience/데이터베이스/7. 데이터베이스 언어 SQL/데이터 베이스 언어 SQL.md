## 1. SQL의 소개
#### SQL의 분류

```SQL
create alter drop
```
- 데이터 정의어(DDL) 
	- 테이블을 **생성**하고 **변경,삭제**하는 기능을 제공 
```sql
insert update delete select
```
- 데이터 조작어(DML) 
	- 테이블에 새 데이터를 **삽입**하거나, 테이블에 저장된 데이터를 **수정,삭제,검색** 하는 기능을 제공 
```sql
revoke grant commit rollback
```
- 데이터 제어어(DCL) 
	- 보안을 위해 데이터에 대한 접근 및 사용 권한을 사용자별로 부여하거나 취소 하는 기능을 제공
